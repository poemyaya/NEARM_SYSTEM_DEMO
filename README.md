# NEARM_SYSTEM_DEMO
### 安装Django 2.1,python 3.5以上版本
### 改ip，路径NEARM_SYSTEM_DEMO/dweb/setting下的ALLOWED_HOSTS = ['172.23.20.198'] 
### 运行命令python3 manage.py runserver 0.0.0.0:8005,确保在NEARM_SYSTEM_DEMO目录下
### 输入文本格式，'Only@Sixteen_e1 is a song by American singer-songwriter Sam@Cooke_e2'：实体之间的空格由@代替，给出e1、e2是谁
### 已运行的句子所得三元组会保存在路径NEARM_SYSTEM_DEMO\static\triple下
